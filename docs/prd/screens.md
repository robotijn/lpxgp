## Marketing

### Landing Page
**Route:** `/` (public)
**Mockup:** `landing.html`

Product marketing page with:
- Hero: "Stop Chasing. Start Matching."
- Problem/solution overview
- Key features (Intelligent Matching, Deep Profiles, AI Insights, Relationship Tools)
- Network animation showing AI capabilities
- How it works (4 steps)
- Trust signals (security, verified data, curated network)
- Primary CTA: "Request Demo"
- Secondary: "Login" link

**Notes:**
- Invite-only platform - no self-registration
- Request Demo leads to contact form
- Login goes to `/login`

### FA Onboard GP Wizard
**Route:** `/fa/onboard-gp`
**Mockup:** `fa-onboard-gp.html`

5-step wizard for GP onboarding:

```
Step 1: Company Details
   - Name, Website, HQ City/Country, Description

Step 2: GP Profile
   - Investment philosophy, Team size, Years investing
   - Typical fund size range, Strategies (multi-select)

Step 3: Access Method
   ( ) Database Only - No users, FA manages
   ( ) Invite Admin - Send invitation email
   ( ) Create User Directly - Set temporary password

Step 4: User Details (if not Database Only)
   If Invite: Email address
   If Direct: Full name, Email, Temporary password

Step 5: Review & Confirm
   - Summary of all entered data
   - [Create GP Organization]
```

---

### FA Onboard LP Wizard
**Route:** `/fa/onboard-lp`
**Mockup:** `fa-onboard-lp.html`

6-step wizard for LP onboarding:

```
Step 1: Organization Details
   - Name, Type (dropdown), Website, HQ City/Country

Step 2: LP Profile
   - Total AUM, PE allocation %, Strategies (multi-select)
   - Check size range, Geographic preferences
   - Mandate description

Step 3: Contacts (Optional, Repeatable)
   - Full name, Title, Email, Phone
   - [x] Decision Maker
   - [+ Add Another Contact]

Step 4: Access Method
   ( ) Database Only - Visible in GP searches, no user login
   ( ) Invite Admin - Send invitation email for LP portal
   ( ) Create User Directly - Set temporary password

Step 5: User Details (if not Database Only)
   If Invite: Email address
   If Direct: Full name, Email, Temporary password

Step 6: Review & Confirm
   - Summary of all entered data
   - Duplicate warning if similar LP exists
   - [Create LP Organization]
```

### Admin - Companies
**Route:** `/admin/companies`

List of all companies with:
- Name, Status (Pending/Active/Deactivated)
- User count, Fund count
- Created date
- Actions: View, Edit, Deactivate

[+ Add Company] button opens company creation modal.

---

### Admin - Company Detail
**Route:** `/admin/companies/[id]`

Single company view with:
- Company info (name, status, created)
- User list (with invite/deactivate actions)
- Fund list
- Activity log
- Settings (plan, limits)

---

### Admin - Users
**Route:** `/admin/users`

All users across all companies:
- Name, Email, Company, Role, Status, Last Login
- Filter by company, role, status
- Search by name/email

---

### Admin - People (Global Contact Database)
**Route:** `/admin/people`

All people in the platform (LP contacts + GP team members):
- Name, Email, Current Org, Title
- Filter by org type (LP/GP), verification status
- Search by name/email
- [+ Add Person] [Merge Duplicates]

---

### Admin - LPs
**Route:** `/admin/lps`

LP organizations:
- Name, Type, AUM, Quality Score, Contact Count
- Filter by type, quality
- Search
- [+ Add LP] [Import]

---

### Admin - LP Detail/Edit
**Route:** `/admin/lps/[id]`

Full LP editor with all fields editable.
Shows linked people with their employment records.

---

### Admin - Data Quality
**Route:** `/admin/data-quality`

- Quality score distribution
- LPs needing review (flagged by users)
- Unverified LPs
- Duplicate candidates
- Review queue with approve/reject actions

---

### Admin - Import Wizard
**Route:** `/admin/import`

Step-by-step import:
1. Upload CSV/Excel
2. Map columns to fields
3. Preview data
4. Validate (show errors/warnings)
5. Approve and import

---

### Admin - System Health
**Route:** `/admin/health`

Service status dashboard:
- Database connection
- Supabase Auth
- OpenRouter API
- Voyage AI
- Error rates (from Sentry)
- Response times
